# Statistical Analysis Scripts

This directory contains scripts used to aggregate and perform statistical analysis on the experimental results generated by the containerized analysis.

## Structure and Execution

The statistical analysis is typically driven by three main scripts in the following order:

1. **merge_data.py**
   - Merges the individual CSV result files from the multiple container outputs (e.g., `results1`, `results2`, etc.) into consolidated datasets.

2. **stats_results.py**
   - Performs the actual statistical computations on the merged data, generating intermediate metrics and statistical tests.

3. **plot_results_v2.py**
   - Reads the statistical metrics and raw data to generate a consolidated PDF report (`statistical_report.pdf`) and other visualizations containing graphs and data distributions.

## How it works within the pipeline

The `run_experiment.sh` script automatically sets up symbolic links to the collected results for a specific dataset (`mds` or `rds`), executes these three scripts sequentially, and then copies the final `statistical_report.pdf` into the structured output directories (`mergedataset/` or `refdataset/`).
